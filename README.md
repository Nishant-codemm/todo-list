# 📝 To-Do List App

A simple and clean To-Do List web application built using HTML, CSS, and JavaScript.

---

## 🚀 Features

* ➕ Add new tasks
* ❌ Delete tasks
* ✅ Mark tasks as completed
* 💾 Save tasks using LocalStorage
* 🔍 Filter tasks (All / Completed / Pending)

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript (Vanilla JS)
